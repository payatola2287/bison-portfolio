== Description ==

Plugin for displaying your projects - could be as a widget, directly in post or page.

== Installation ==

Installing "Bison Portfolio" can be done either by searching for "Bison Portfolio" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
